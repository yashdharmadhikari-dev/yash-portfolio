Interactive 3D Bot with Generative AI

(B.E. Capstone Project)

Description: The integration of 3D technology and artificial intelligence (AI) is revolutionizing the way that immersive and interactive user experiences are created in today's digital world. The goal of this research is to create a dynamic and engaging user interface by utilizing the complementary abilities of three-dimensional environments and generative AI models...

  • Developed the entire backend pipeline using Python sockets for seamless interaction between Unity 3D and external APIs, reducing response time by 40%.
  • Created and implemented a face recognition model using FaceNet, achieving an accuracy improvement of 92% for real-time recognition tasks in Unity 3D applications.
  • Optimized the model's performance by reducing the model's weight through pruning and quantization techniques, decreasing processing time by 30% while maintaining accuracy.
  • Integrated the FaceNet model with Unity's input system, enabling real-time face detection and recognition within interactive 3D environments, boosting interaction efficiency by 50%.
  • Collaborated with API services to ensure smooth data flow and interaction between Unity 3D and the backend server, increasing data synchronization efficiency by 35%.

Video Summary using AI

(T.E Capstone Project)

Description: The Video Summary using AI is a cutting-edge web application designed to automatically generate concise and accurate summaries of long videos. By leveraging advanced models like the Bi-modal Transformer (BMT) and OpenAI's Whisper model, the system provides seamless and efficient video summarization, making content more digestible and time-efficient for users. This application is ideal for summarizing educational videos, corporate meetings, webinars, and more, significantly reducing the time users need to extract key insights from video content.

  • Developed the core architecture for real-time video analysis and transcription, reducing manual processing time by 60%.
  • Utilized OpenAI's Whisper model to achieve a transcription accuracy of 94%, ensuring high-quality summaries even for noisy environments.
  • Implemented the Bi-modal Transformer (BMT) for multi-modal data processing, combining video and audio inputs, improving summarization accuracy by 87% compared to traditional text-based methods.
  • Optimized the BMT model to handle large video files (over 2 hours) with a 45% reduction in memory usage, ensuring smooth and fast performance on web browsers.
  • Integrated automatic language detection, expanding the application's usability across 15 different languages, with an accuracy improvement of 90%.
  • Collaborated with cloud storage services to ensure scalable video processing, improving data retrieval speed by 50%.

Smart Music Player

(Android Application)

Description: The Smart Music Player is an Android-based application designed with accessibility in mind, particularly for people with physical disabilities. It leverages Google Voice-to-Text API to enable hands-free control, making it easier for users to interact with the player using voice commands. This project, aimed at improving accessibility in everyday technology, became the runner-up at the National Hackathon 2021. The music player supports standard functionalities such as play, pause, resume, and file browsing, allowing disabled users to enjoy an inclusive digital experience with ease.

  • Developed the application using Android Java, ensuring a smooth user experience and responsive design, increasing accessibility for disabled users by 60%.
  • Integrated Google’s Voice-to-Text API to allow full control via voice commands, achieving 95% command accuracy in noisy environments, enhancing usability in diverse settings.
  • Reduced manual interaction by 70%, enabling users to perform key actions such as play, pause, and skip through voice control.
  • Implemented background music playing functionality, allowing users to seamlessly browse files while music continues, enhancing multitasking efficiency by 50%.
  • Optimized the app’s memory usage by 30%, ensuring smooth performance even on low-end devices, making it accessible to a wider user base.
  • Achieved runner-up position at the National Hackathon 2021 by demonstrating innovative use of voice control technology for inclusivity in digital media consumption.
